## InterMaritime

<Img 
  fluid={props.data.intermaritime.childImageSharp.fluid}
  className="medium-image float-right small-shadow image-gap"
  alt='InterMaritime Website'
/>

[InterMaritime.org](https://intermaritime.org/) Es el sitio web corporativo del Grupo de Empresas InterMaritime, comprendidas por 5 entidades. Construí este sitio creando una plantilla personalizada de WordPress. Luego la convertí a un sitio web estático utilizando un plugin llamado WP2Static. El sitio está hospedado en una cubeta de S3 en Amazon Web Services (AWS). El costo de hospedaje es muy bajo, y la carga del sitio es muy rápida. La misma cuenta con un formulario de contacto el cual se conecta con una función lambda en AWS, de esta manera, el formulario puede enviar correos (entre otras tareas) sin la necesidad de levantar un VPS.

## ICS class

<Img 
  fluid={props.data.icsclass.childImageSharp.fluid}
  className="medium-image float-left small-shadow image-gap"
  alt='ICSClass Website'
/>

[ICSClass.org](https://icsclass.org/) Es un sitio web de la empresa más importante del grupo InterMaritime. Este sitio web lo construí con un tema personalizado, creado por mi persona. El tema utiliza el starter [underscores](https://underscores.me/). El sitio cuenta con varias características como buscadores de entradas personalizadas entre otras. El sitio cuenta con un sistema de análisis de visitantes, un formulario de contáctenos personalizado, y un área para editar noticias y eventos. El sitio está hospedado en un servidor virtual privado (VPS). Todos los cambios al código del sitio se gestionan a través de git, se empujan a un repositorio privado en bitbucket, y luego a un repositorio limpio en el VPS.

</VisitButton>

## WordCamp Panamá 2019

<Img 
  fluid={props.data.wordcamp.childImageSharp.fluid}
  className="medium-image float-right small-shadow image-gap"
  alt='WordCamp Panama 2019 Website'
/>

Este sitio fue creado para promocionar el evento que se daba por primera vez en la ciudad de Panamá (y en todo el país). Fue creado con muchas limitaciones de parte del organizador del evento WordCamp Central, quienes regulan todo lo que se puede o no se puede hacer en este sitio. El sitio sólo podía actualizarse utilizando un tema prefabricado o un tema totalmente desnudo. Utilicé el tema desnudo para poder estilizarlo a mi propia manera. Usando técnicas avanzadas de CSS pude hacer muchas variantes al sitio para que este fuera muy diferente a todos los sitios para WordCamp en el mundo. Este sitio se puede utilizar como plantailla para otros sitios. Es definitivamente único en su clase.
